# Run Analysis Skill

## Pace Split Analysis

- Calculate mean pace across all km splits
- **Positive split**: second half slower than first half (common, usually bad for racing)
- **Negative split**: second half faster (ideal for half-marathon racing)
- **Even split**: <2% variation (good for steady-state runs)
- Flag any km where pace deviates **>15%** from the mean — likely a walk break, stop, or terrain issue
- For interval workouts, analyze work vs rest segments separately

## HR Drift / Cardiac Decoupling

Formula: `(avg_HR_second_half - avg_HR_first_half) / avg_HR_first_half * 100`

- **<5%**: good aerobic fitness for that pace
- **5-10%**: moderate drift, acceptable for hard efforts, concerning for easy runs
- **>10%**: significant drift — possible dehydration, overheating, or pace too aggressive
- Compare at similar paces only — exclude warm-up and cool-down

## Cadence Benchmarks

- **Optimal half-marathon**: 170-180 spm
- **<165 spm**: likely overstriding, higher injury risk, less efficient
- **>185 spm**: may indicate shuffling or very short stride
- Watch for cadence drop in final third — fatigue signal
- Cadence should stay relatively stable regardless of pace changes

## Running Power Efficiency

- Power-to-pace ratio: watts per min/km
- Consistent power + rising HR = cardiac drift (aerobic limitation)
- Rising power + stable pace = uphill terrain
- Dropping power + same HR = fatigue / form breakdown
- Expected ranges vary by athlete; track trends over sessions

## Ground Contact Time

- **200-250ms**: normal range for recreational runners
- **<200ms**: elite-level efficiency
- **>280ms**: inefficient, possible overstriding
- Watch for GCT increasing in final third — fatigue and form degradation
- Left-right balance: >2% asymmetry may indicate compensation

## HR Zone Targets by Workout Type

| Workout Type | Primary Zones | Notes |
|---|---|---|
| Easy / Recovery | Z1-Z2 | >80% of time in Z1-Z2, flag if Z3+ |
| Long Run | Z1-Z2 | Some Z3 acceptable in final km |
| Tempo | Z3-Z4 | Main set in Z3-Z4, warm-up/cool-down Z1-Z2 |
| Intervals | Mixed | Work bouts Z4-Z5, recovery Z1-Z2 |
| Race Pace | Z3 | Sustained Z3 for half-marathon effort |

## Heat & Humidity Adjustments

- **>25C**: expect +5-10 bpm at same pace per 5C above 25C
- **>28C + >60% humidity**: significant compound effect, pace should slow 5-10%
- **<10C**: slower warm-up expected, may take 10-15 min to reach target HR
- Adjust expectations accordingly — a "slow" run in 35C heat may actually be a strong effort
